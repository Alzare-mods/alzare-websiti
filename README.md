# ALZARÉ - Sitio listo para GitHub Pages

Proyecto listo para subir a GitHub Pages. Estructura simple y profesional (negro + dorado), estilo similar a una landing de marca deportiva.

## Instrucciones rápidas para publicar en GitHub Pages

1. Crea un repositorio público llamado `alzare-website` (o el que prefieras).
2. Sube todo el contenido de la carpeta del proyecto (index.html, productos.html, styles.css, script.js y la carpeta `img`). Puedes hacerlo con "Upload files" en la interfaz de GitHub.
3. En `Settings` → `Pages`, selecciona la rama `main` y la carpeta `/ (root)`. Guarda.
4. GitHub Pages publicará tu sitio en `https://TU_USUARIO.github.io/NOMBRE_DEL_REPO` (puede tardar unos minutos).
